# 🌱 Smart Rooftop Farming Advisor for Urban Areas

An AI-powered web application designed to assist urban farming decisions through 
**crop recommendation, fertilizer recommendation and plant disease detection with pesticide recommendation** using 
Machine Learning and Deep Learning models.


# 🚀 Features

# 🌾 Crop Recommendation
- Recommends best suitable crop based on:
  - Soil type
  - location
  - Month / season
  - Soil nutrients (N, P, K, pH)
  - Real-time weather data (API-based)
- Powered by **XGBoost ML model**

# 💧 Fertilizer Recommendation
- Suggests optimal fertilizer based on:
  - Crop type
  - Soil type
  - Growth stage
  - NPK values and pH
- Improves yield and soil health

# 🦠 Plant Disease Detection
- Detects tomato leaf diseases using image upload
- Powered by **CNN (TensorFlow/Keras)**
- Outputs:
  - Disease name
  - Confidence score
  - Treatment suggestions

# 🌦️ Weather Integration
- Live weather data using OpenWeatherMap API
- Includes:
  - Temperature
  - Humidity
  - Rainfall
  - Sunlight estimation

# 🌿 Smart Watering Advice
- Suggests irrigation based on:
  - Soil type
  - Weather conditions
  - Crop type



# 🧠 Tech Stack
 
- **Programming Language:** Python  
- **Machine Learning:** XGBoost, Scikit-learn  
- **Deep Learning:** TensorFlow / Keras  
- **Libraries:** Pandas, NumPy, Joblib, Pillow  
- **API:** OpenWeatherMap  
- **Model Storage:** Pickle / Joblib
